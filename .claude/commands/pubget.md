# Flutter 의존성 설치

Flutter 패키지 의존성을 설치합니다.

## 사용법
```
/pubget
```

## 실행 명령

```bash
flutter pub get
```

## 추가 명령

### 의존성 업그레이드
```bash
flutter pub upgrade
```

### 특정 패키지 추가
```bash
flutter pub add [package_name]
```

### 개발 의존성 추가
```bash
flutter pub add dev:[package_name]
```

## 현재 프로젝트 주요 의존성

- **dio**: HTTP 클라이언트
- **flutter_secure_storage**: 보안 저장소
- **kakao_flutter_sdk_user**: 카카오 로그인
- **google_sign_in**: Google 로그인
- **sign_in_with_apple**: Apple 로그인
- **webview_flutter**: WebView (결제)
- **flutter_local_notifications**: 로컬 알림
